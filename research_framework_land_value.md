# Shanghai Millionaire: Land Value & Game Board Price Correlation
# 上海大富翁：地价与棋盘价格关联研究框架

## Research Core Question 核心研究问题

> **Does the Shanghai Millionaire game board encode colonial spatial hierarchy rather than empirical land values?**
> 上海大富翁棋盘是否编码了殖民地空间等级，而非实证地价？

---

## 1. Theoretical Framework 理论框架

### 1.1 Colonial Spatial Hierarchy Theory 殖民地空间等级理论
- **Henri Lefebvre** (The Production of Space): Space as social product, not neutral container
- **Wen-hsin Yeh** (Shanghai Splendor): Colonial consumption culture and spatial stratification
- **Key concept**: "Colonial gaze" — whose perspective defines "valuable" space?

### 1.2 Game Studies Perspective 游戏研究视角
- **Mary Flanagan**: Games as cultural artifacts encoding social values
- **Miguel Sicart**: Values embedded in game mechanics
- **Research gap**: How does localization (London → Shanghai) transplant value systems?

### 1.3 Economic Geography 经济地理学
- **Bid-rent theory**: Land value as function of distance from center + social status
- **Shanghai exception**: Multiple competing centers (British Settlement, French Concession, Chinese City)

## 2. Empirical Research Questions 实证研究问题

### Q1: Price Correlation Analysis 价格关联分析
**Hypothesis**: Game prices correlate with colonial prestige zones, not market rents.

| Game Price Tier | Streets | Geographic Zone | Colonial Status | Expected vs. Actual |
|----------------|---------|-----------------|-----------------|---------------------|
| $350-400 (Deep Blue) | Bund, Kiukiang | International Settlement Core | British commercial elite | **Expected: High** |

| $260-280 (Yellow) | Bubbling Well, Great Western, Yu Yuen | IS West / Residential | British residential | **Expected: Medium** |
| $220-240 (Red) | Seymour, Hardoon | IS West | Mixed residential | **Expected: Medium** |

**Key Anomaly to Investigate**: 
- Ningpo/Tientsin/Honan are geographically central but game-priced low
- Are these "Chinese commercial streets" systematically undervalued in colonial valuation?

### Q2: Temporal Dimension 时间维度
**Research Question**: How did 1937-1941 "Solitary Island" period affect land values vs. game encoding?

- Pre-war (pre-1937): Colonial hierarchy stable
- War period (1937-1941): Refugee influx, Chinese areas appreciated
- Game likely printed: 1938-1940
- **Hypothesis**: Game freezes pre-war colonial imagination, not wartime reality

### Q3: Comparative Analysis 比较研究
**Research Question**: How does Shanghai localization differ from other Monopoly localizations?

| Version | Localization Type | Value System Source | Key Distinction |
|---------|-------------------|---------------------|-----------------|
| Atlantic City (1935) | Original | Real estate values + designer adjustment | Baseline |
| London (1936) | National | Real neighborhood prestige | Direct mapping |
| Berlin (1936) | National (later banned) | Nazi-era spatial politics | Political manipulation |
| Shanghai (c.1938) | **Colonial** | Colonial racial/economic hierarchy | **Power asymmetry** |

**Shanghai uniqueness**: Only major version where localization involves colonial power structure, not just geographic translation.

## 3. Data Sources & Methodology 数据来源与方法

### 3.1 Primary Sources 一手资料

#### A. Shanghai Municipal Archives 上海市档案馆
- **工部局年报 (SMC Annual Reports)**: 1935-1941
  - Property tax assessments by district
  - Land value indices
  - Access: archives.sh.cn (partially digitized)

#### B. Digitized Newspapers 数字化报纸
- **North China Daily News** (ProQuest Historical)
  - Real estate advertisements
  - Rental market reports
  - Search: "rent," "property," "real estate" + street names

- **The China Press, Shanghai Times**
  - Cross-reference for consistency

#### C. Academic Datasets 学术数据集
- **Virtual Shanghai** (virtualshanghai.net)
  - Historical maps with land use overlays
  - Some property data available

- **北京大学历史地理研究中心**
  - Shanghai 1930s land use GIS (contact for access)

### 3.2 Secondary Sources 二手文献

#### Key Scholars 关键学者
1. **Christian Henriot** (Lyon/Virtual Shanghai)
   - *Shanghai, 1927-1937: Municipal Power, Locality, and Modernization*
   - Expert on SMC governance and urban space

2. **Wen-hsin Yeh** (UC Berkeley)
   - *Shanghai Splendor: Economic Sentiments and the Making of Modern China*
   - Colonial consumption culture

3. **Toby Lincoln**
   - *Urbanizing China*
   - Republican-era urban development

4. **Nara Dillon** (on refugee camps)
   - Context for 1937-1941 population shifts

#### Comparative Game Studies 游戏研究比较
- **Ralph Anspach** (*The Billion Dollar Monopoly Swindle*)
  - Atlantic City land value research
  - Legal documents on Monopoly history

- **Philip E. Orbanes** (*Monopoly: The World's Most Famous Game*)
  - Official Hasbro historian, but useful baseline

### 3.3 Methodology 研究方法

#### Step 1: Data Collection 数据收集
- Extract game prices and street coordinates (✓ Completed)
- Map streets onto 1930s Shanghai base map (✓ Completed)
- Collect contemporary land value data from SMC reports
- Compile rental advertisements from newspapers

#### Step 2: Correlation Analysis 关联分析
- Create price index: Game price vs. empirical land value
- Identify outliers (streets where game price ≠ market value)
- Categorize outliers by colonial/racial factors

#### Step 3: Qualitative Analysis 定性分析
- Colonial discourse analysis: How are streets described in English-language sources?
- Compare Chinese vs. English naming conventions
- Analyze "Avenue" vs. "Road" naming patterns

#### Step 4: Temporal Comparison 时间比较
- Track land value changes 1935-1941
- Compare game encoding to pre-war vs. wartime values
