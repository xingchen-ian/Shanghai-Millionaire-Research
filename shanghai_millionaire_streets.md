# Shanghai Millionaire - 街道名称整理

## 项目概述

## 历史背景说明

### 命名特点
1. **人名路名**：多以外国军政人物命名（如霞飞路、海格路、贝当路）
2. **中国地名**：部分以中国城市命名（如南京路、北京路）
3. **本地商人**：个别以在沪知名商人命名（如虞洽卿路、哈同路）

### 租界分区
- **公共租界**：南京路、北京路、外滩等区域
- **法租界**：霞飞路、贝当路、亚尔培路等区域

### 路名变迁
1943年后，汪伪政府收回租界，大量路名被改为中国地名或革命名称。例如：
- 霞飞路 → 泰山路 → 林森路 → 淮海路
- 海格路 → 华山路
- 静安寺路 → 南京西路
