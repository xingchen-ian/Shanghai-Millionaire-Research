# Shanghai Millionaire 印刷来源考证：1930-40年代上海有能力印刷外文桌游的印刷厂与出版社

> 研究日期：2026-04-17  
> 研究者：Ian  
> 核心问题：Shanghai Millionaire（上海大富翁盗版）的印刷厂是谁？

---

## 一、已知线索汇总

| 线索 | 来源 | 可靠性 |
|------|------|--------|
| 游戏基于英国Waddington's伦敦版设计 | World of Monopoly (Albert Veldhuis) | 高 |
| 估计生产于1930年代末或1940年代初 | rickshaw.org (Ralph Harpuder) | 中 |
| 在上海**大型百货公司**销售，明确提及**永安百货(Wing On)** | rickshaw.org | 高 |
| 商业化大规模生产（非小规模手工制作） | Kent McKeever（上海历史学家）分析 | 中 |
| 印刷精良，48.5×48.5cm，全英文 | Ian实物观察 | 高 |
| 原拥有者家庭二战前/期间从东欧逃往上海 | Gary Korzenstein（收藏者） | 高 |
| 游戏板背面有黑色底面+白色标签 | World of Monopoly描述 | 中（标签内容未知） |
| 制造商不详（Unknown） | World of Monopoly | — |

**关键推论**：该游戏需要彩色印刷能力（棋盘+纸币+地契卡）、纸盒制作能力、英文排版能力，以及面向外侨市场的分销渠道。

## 二、有能力印刷外文桌游的印刷厂/出版社候选名单

### 第一梯队：外资/外文印刷企业（可能性最高）

#### 1. 别发洋行 / 别发印书局（Kelly & Walsh Limited）

| 项目 | 详情 |
|------|------|
| **英文名** | Kelly & Walsh Limited |
| **中文名** | 别发洋行、别发印书局、别发印书馆 |
| **成立** | 1876年（由Kelly and Co.与F. & C. Walsh合并） |
| **地址** | 上海外滩11号 → 1918年迁至南京路 |
| **国别** | 英商 |
| **业务范围** | 图书销售与出版、高品质印刷、文具、石印(lithographer) |
| **外文能力** | 上海最早主营西文书的企业，印刷质量极高 |
| **产品** | 英文书籍、风景明信片、旅游纪念册（如*Shanghai of To-Day*系列） |
| **活跃期** | 1870s-1940s（日本占领后迁港） |
| **印刷能力** | 承印"各类高品质产品"（every description of high-quality printing） |

**可能性评估**：⭐⭐⭐⭐  
**理由**：上海最知名的外文印刷出版商，具备彩色印刷和英文排版能力。但主要业务方向是书籍和明信片，尚未发现涉足游戏产品的记录。需查阅其1930年代商业记录确认。  
**查证建议**：查阅上海工部局档案中Kelly & Walsh的商业注册信息；搜索北华捷报(Kelly & Walsh自身有利益关联)广告栏。

#### 2. 英商美灵登有限公司（Millington Limited）

| 项目 | 详情 |
|------|------|
| **英文名** | Millington Limited |
| **中文名** | 美灵登广告公司、美灵登有限公司 |
| **成立** | 1927年，由Francis C. Millington创办 |
| **地址** | 上海南京路创办 → 后迁至香港路（中央车库/Central Garage旧址） |
| **国别** | 英商 |
| **业务范围** | 广告代理、商业印刷、出版 |
| **外文能力** | 英商企业，全英文运营 |
| **产品** | 户外广告、商业印刷品、出版了《上海之户外广告》一书 |
| **关联** | 汇丰银行为其股东；上海知名英商广告印刷公司 |
| **后续** | 1930年代原址改为柳荫印刷所(1940s) |

**可能性评估**：⭐⭐⭐⭐⭐  
**理由**：英商广告印刷公司，具备彩色商业印刷能力，且是面向外侨市场的服务商。作为广告公司，承接桌游印刷这样的商业订单完全在其业务逻辑内。1937年该公司还发行股票（汇丰银行承购），说明经营规模可观。  
**查证建议**：查阅美灵登公司1937-1940年间的商业印刷订单记录；搜索北华捷报中Millington的广告。

#### 3. 字林西报馆印刷部（North-China Daily News / North China Herald）

| 项目 | 详情 |
|------|------|
| **英文名** | North-China Daily News (NCDN) |
| **中文名** | 字林西报、字林洋行 |
| **成立** | 1850年（北华捷报）→ 1864年改为日报 |
| **地址** | 上海公共租界 |
| **国别** | 英商 |
| **业务范围** | 英文报纸出版、商业印刷服务 |
| **印刷能力** | 拥有大型印刷厂，承印各类商业印刷品 |
| **活跃期** | 1850-1951（1941-45年 wartime break） |

**可能性评估**：⭐⭐⭐  
**理由**：上海最大的英文报纸出版商，拥有强大的印刷设备。但1941年被日军停刊，与Shanghai Millionaire的生产时间窗口（约1938-1940）有重叠。报纸印刷厂偶尔承接商业印刷订单是常见做法。  
**注意**：1941-1945年停刊，若游戏生产于此期间则排除。

### 第二梯队：华资大型印刷企业（有可能但需进一步确认）

#### 4. 商务印书馆（The Commercial Press）

| 项目 | 详情 |
|------|------|
| **英文名** | The Commercial Press |
| **成立** | 1897年 |
| **地址** | 上海宝山路（总厂） |
| **国别** | 华商 |
| **业务范围** | 出版、印刷、教科书、文具 |
| **印刷能力** | 上海最大印刷厂，拥有胶版印刷、石印、凹印等全套设备 |
| **外文能力** | 出版英文教科书，与美商Macmillan合作20年 |
| **重大事件** | 1932年1月29日被日军轰炸，总厂及46万册古籍被毁 |

**可能性评估**：⭐⭐⭐  
**理由**：上海最大的印刷企业，1932年后虽遭重创但逐步恢复。具备英文排版和彩色印刷能力。但主要面向华文市场，盗版外文桌游不太符合其商业模式。  
**注意**：1937年八一三事变再次受损，1938-1940年期间生产能力可能有限。

#### 6. 上海粹华卡片厂

| 项目 | 详情 |
|------|------|
| **成立** | 1919年8月 |
| **创始人** | 黄涤生 |
| **地址** | 上海郑家木桥（今福建中路） |
| **业务范围** | 扑克牌、各类卡片、证书 |
| **关键发展** | 1931年从德国引进设备，正式生产扑克牌 |
| **重大事件** | 1937年日本侵占上海后停产 |

**可能性评估**：⭐⭐⭐⭐  
**理由**：**这是最直接的候选！** 粹华卡片厂是中国第一家扑克牌制造厂，专门生产纸质游戏用品（扑克牌、卡片等），产品定位与Shanghai Millionaire完全吻合。1931年开始生产扑克牌，1937年停产——如果游戏生产于1937年前，则时间窗口匹配。但1937年后停产是个问题。  
**关键问题**：粹华卡片厂是否承接过英文桌游的代工印刷？其"马戏丑角"扑克牌已证明具备彩色纸牌印刷能力。  
**查证建议**：查阅粹华卡片厂1931-1937年的产品目录和客户记录。

### B. 具体查证步骤

1. **下载行名辞典Excel** → 筛选所有printing/print相关条目 → 建立完整印刷厂名录
2. **查阅美灵登公司(Millington Ltd.)档案** → 确认其印刷业务范围是否包括游戏/纸制品
3. **查阅粹华卡片厂产品目录** → 确认是否有英文游戏代工记录
4. **追踪永安百货进货渠道** → 查看是否有进口游戏或本地印刷品的进货记录
5. **检查游戏板背面白色标签** → 若有文字可能直接指向印刷厂
6. **搜索1938-1940年北华捷报广告** → 可能找到Shanghai Millionaire的零售广告
7. **联系Kent McKeever** → 他在World of Monopoly页面分析了生产规模，可能掌握更多线索

### C. 新发现的研究线索

- **美灵登画校**：杨永康（1932年生）17岁开始"在上海美灵登画校专攻油画"——说明Millington公司不仅有印刷业务，还设有艺术培训部门，与其广告/印刷业务高度一致
- **美灵登公司原址变迁**：南京路创办 → 香港路中央车库 → 1940年代改为柳荫印刷所——公司本身可能经历了结构性变化
- **1937年美灵登股票**：汇丰银行承购1000股（每股4元），说明公司规模不小

## 四、初步结论与假设排序

### 印刷厂假设排序（从最可能到最不可能）

| 排序 | 候选 | 核心理由 |
|------|------|----------|
| 1 | **美灵登公司 (Millington Ltd.)** | 英商广告印刷公司，规模大，面向外侨市场，具备彩色商业印刷能力 |
| 2 | **粹华卡片厂** | 专门生产扑克牌/纸牌的中国工厂，产品类型直接匹配 |
| 3 | **别发洋行 (Kelly & Walsh)** | 上海最大外文印刷出版商，印刷质量匹配 |
| 4 | **字林西报馆印刷部** | 大型英文印刷设备，可能有商业印刷副业 |
| 5 | **商务印书馆** | 最大印刷厂但战争受损，商业模式不太匹配 |
| 6 | 池田印刷/日资厂 | 有彩印能力但日英关系紧张 |
| 7 | 俄侨/犹太难民印刷 | 规模太小，不太可能 |

### 关键时间窗口问题

- 若游戏生产于**1937年前**：所有候选均有可能，粹华卡片厂时间窗口最匹配
- 若游戏生产于**1937-1941年**：日资厂可能性降低，粹华停产，外资厂（美灵登、Kelly & Walsh）可能性上升
- 若游戏生产于**1941-1945年（日军占领期）**：字林西报已停刊，Kelly & Walsh迁港，可能性集中在仍运营的日资或中立资本印刷厂
